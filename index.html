Music API

A Music API é uma aplicação desenvolvida pela estudante Maria Clara Farias Barbosa. Essa aplicação possui o intuito de que o usuário pesquise por diversos artistas do ramo musical, seja cantor(a) ou banda. Ao realizar a pesquisa, o usuário receberá informações do artista, como: Gênero musical e Músicas onde o usuário será encaminhado para a plataforma “Vagalume”, assim, podendo escutar a música selecionada.
